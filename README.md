# GitHub Demo Project

Welcome to our **GitHub Demo Repository!**  
This project is created to practice **GitHub collaboration** using push commands.

---

## Project Description
This repository serves as a **demo project** where all team members are required to:
- Practice using GitHub
- Push their own changes
- Collaborate in a shared repository

---

##  Tools Used
The following tools and technologies are used in this project:

- Git
- GitHub
- Linux Terminal
- Acode
- Markdown (.md)
- Any programming language or text (HTML, CSS, JavaScript, txt, etc.)

---

##  Team Instructions
Each team member must:

```bash
# 1. Clone the repository
git clone https://github.com/your-repo-link.git

# 2. Go to project folder
cd your-project-folder

# 3. Make changes

# 4. Add changes
git add .

# 5. Commit changes
git commit -m "Your message"

# 6. Push changes
git push

---

##  Objective
The goal of this activity is to:
- Understand how GitHub works
- Practice version control using Git
- Experience real team collaboration

## What You Can Add
You are free to push:
- Simple code (HTML, CSS, JS, TXT, etc.)
- Notes or text files
- Practice projects
- Anything related to learning GitHub
